# Tiny Stopwatch

> A tiny stopwatch that lives in your Chrome toolbar.

ポップアップすらない、Chromeツールバー常駐のミニマムなストップウォッチです。  
拡張機能のアイコンそのものが分・秒のデジタル表示になります。

## 使い方

- **左クリック**：スタート
- **もう一度左クリック**：ストップ
- **停止中に左クリック**：続きから再開
- **マウスホバー**：`時:分:秒.0.1秒` の詳細な経過時間を確認
- **右クリック → `Reset to 00:00`**：停止してゼロにリセット
- **右クリック → `Theme`**：`Dark` / `Light` を切り替え

アイコンは、上段が「分」、下段が「秒」です。

```text
MM
SS ·
```

右下の小さなドットは動作中ランプです。計測中だけ、0.5秒点灯・0.5秒消灯で鼓動します。

ホバー表示は時間を先頭にした次の形式です。

```text
00:12:34.5 — Tiny Stopwatch — running; click to stop
```

停止中も0.1秒まで保持するため、ストップした瞬間の値をホバーで確認できます。

### 60分を超えたとき

アイコン表示は従来どおり60分周期です。`59:59` の次は `00:00` に戻って表示を続けます。  
一方、内部の経過時間はリセットされません。たとえば1時間27分43.6秒なら、アイコンは `27 / 43`、ホバーは `01:27:43.6` と表示します。

> Chromeは拡張機能アイコンの右クリックを標準メニューに使用するため、リセットや配色切替は右クリック後にメニュー項目を選択します。

## インストール

Chromeウェブストアを使わず、ローカルの「パッケージ化されていない拡張機能」として読み込みます。

1. このリポジトリの **Code → Download ZIP** を選ぶか、Gitでクローンします。
2. ZIPの場合は任意の場所に展開します。
3. Chromeで `chrome://extensions` を開きます。
4. 右上の **デベロッパー モード** をオンにします。
5. **パッケージ化されていない拡張機能を読み込む** をクリックします。
6. `manifest.json` が入っている `tiny-stopwatch` フォルダを選択します。
7. Chromeの拡張機能メニューから **Tiny Stopwatch** をツールバーに固定します。

## 更新

リポジトリを再度ダウンロードするか `git pull` したあと、`chrome://extensions` の **Tiny Stopwatch** にある再読み込みボタンを押してください。

既に計測中の場合、再読み込み前の状態は `chrome.storage.local` から復元されます。

## 仕様

- アイコン表示：`00:00`〜`59:59`を60分周期で表示
- ホバー表示：`HH:MM:SS.t`（0.1秒単位）
- 経過時間：60分を超えて保持
- テーマ：Dark / Light
- ポップアップなし
- Webページへのアクセス権限なし
- 外部通信なし
- 状態とテーマは `chrome.storage.local` にのみ保存
- Manifest V3
- Chrome 110以降

計測中に拡張機能のService Workerが再起動しても、保存した開始時刻から経過時間を復元します。
